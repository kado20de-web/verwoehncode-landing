# E-Book-Dateien

Lege hier die fertigen PDF-Dateien ab. Die Dateinamen **müssen exakt** mit
`lib/products.js` übereinstimmen:

- `101-romantische-ideen.pdf`
- `massagetechniken-fuer-paare.pdf`
- `so-werdet-ihr-ein-musterpaar.pdf`

Diese Dateien werden nach erfolgreichem Kauf über zeitlich begrenzte,
signierte Links ausgeliefert (`/download?...`) und sind nicht öffentlich
direkt erreichbar.
